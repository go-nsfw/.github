# Go NSFW

*A collection of Go libraries for enterprise software (North-South Firewall).*

Relax. It stands for North-South Firewall. We build Go libraries for the traffic that crosses trust boundaries: ingress, egress, and everything the perimeter touches. If your architecture has a DMZ, we have opinions about it.
